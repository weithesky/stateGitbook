gitbook

gitBook入门

配置和目录简洁，但不易私人定制化

  • gitbook安装

    npm install -g gitbook-cli

  • gitbook初始化项目

    gitbook init

    生成README.md为样本文件

    生成SUMMARY.md为目录文件

    可以手动创建book.json配置文件

  • gitbook运行

    #默认4000端口
gitbook serve

#指定端口
gitbook serve --port 2333

  • gitbook编译

    gitbook build
