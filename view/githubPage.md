- ###### git操作

  ```shell
  # 本地初始化git
  git init
  
  # clone远程仓库到本地
  git clone 远程仓库地址
  
  # 本地要提交的文件复制到仓库目录下
  
  # 新文件添加到git
  git add .
  
  # 提交
  git commit -m '提交信息'
  
  # push到远程仓库
  git push
  ```
