### vue3.0

> ```shell
> #安装vue3.0
> npm install -g @vue/cli
> 
> #创建项目
> vue create vue-app
> 
> #使用vue-gui
> vue ui
> 
> 
> ```



### vue可选配置文件

> 在package.json同级目录创建**vue.config.js**,如果存在会被`@vue/cli`自动加载。



### 配置淘宝镜像cnmp

> ```shell
> npm install -g cnpm --registry=https://registry.npm.taobao.org
> ```

> [淘宝镜像，可以下载某些软件](https://npm.taobao.org/)
